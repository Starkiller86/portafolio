# Portafolio Personal - Mario Alberto Hernández Salinas

Sitio web de portafolio personal con diseño retro 8-bit usando HTML, CSS y Bootstrap.

## Estructura

```
portafolio/
├── index.html    # Página principal
├── styles.css    # Estilos CSS
└── README.md     # Este archivo
```

## Tecnologías

- HTML5
- CSS3
- Bootstrap 5 (CDN)
- Google Fonts - Press Start 2P
- Font Awesome 6 (íconos)
- Devicons (íconos de tecnologías)

## Características

- Diseño retro 8-bit con tonos claros
- Navbar fija con scroll suave
- Responsive (móvil, tablet, desktop)
- Botones copiar email/teléfono al portapapeles
- Hover effects en cards y botones
- Secciones: Hero, Sobre mí, Proyectos, Habilidades, Contacto

## Uso

Abre `index.html` en cualquier navegador web. No requiere servidor ni dependencias adicionales.
